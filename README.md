# 🌐 Online Downloadable CV

A modern interactive CV web project built using **HTML, CSS, and JavaScript**, featuring **light/dark mode toggle** and a **download option** for the CV.

---

## ✨ Features

- 🌞 Light / 🌙 Dark mode toggle (sun icon switch)
- 📄 Download CV as image/PDF
- 🎨 Clean and responsive UI
- ⚡ Simple and fast frontend project
- 💻 Built using HTML, CSS, JavaScript

## 🚀 How It Works

- Click the **sun icon** to switch between light and dark mode 🌗  
- Click the **download button** to download the CV 📥  
- Works smoothly on desktop and mobile 📱
